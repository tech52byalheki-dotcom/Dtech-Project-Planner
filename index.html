Customer Places an Order
Browse → Cart → Pay → Track
1
Browse products via dtechmobileapp:3000
2
Add to cart → stored in Redis
3
Razorpay payment → order confirmed in MySQL
4
Admin assigns driver → push notification sent
5
Driver delivers → order DELIVERED
📦
Vendor Restock Order
Login → Order → Approve → Manifest
1
Vendor logs in via /common/v1/validateLogin
2
Create product order → order_details + order_products in DB
3
Admin approves → inventory updated at warehouse
4
Vendor generates PDF manifest via Puppeteer

Vendor Registration & Verification
Register → Submit Docs → Admin Approves → Active
1
Vendor submits profile with GST + trade licence files
2
Admin reviews vendor profile & documents
3
Admin verifies → isDocumentVerified = true
4
Vendor active → can log in and place orders
